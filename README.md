# copilotstats
Tired of checking usage on the official site or VS Code settings.

Reversed the internal API to fetch usage stats directly via GitHub tokens.

Displays more details than the official UI.

## Features
Token Gen: Auto-extracts Github Copilot tokens.

Usage Check: More accurate and readable than the official one.

Minimal: No bloat.

## Quick Start
```Bash
git clone https://github.com/uheej0625/copilotstats
cd copilotstats
npm install
npm start
```
